# Employee Management System

A Spring Boot REST API for managing employees with role-based access control.

## Features
- Employee CRUD Operations
- Spring Security Authentication
- Role Based Authorization (ADMIN / USER)
- BCrypt Password Encryption
- H2 Database
- REST APIs

## Tech Stack
- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- H2 Database
- Maven

## API Endpoints

### Authentication
POST /auth/register

### Employees
GET /employees
POST /employees
PUT /employees/{id}
DELETE /employees/{id}
